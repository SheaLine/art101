Lab 13 - Loops

Challenge

The challenge of this lab is to use a loop to map a phrase to a certain number and print the results to the page.

Problems

There was a lot of trial and error with this lab. Using console.log() and the browser debug tools helped me see what exactly was happening under the hood.

Reflection

This lab was a little more complex than the other labs, but it was still fairly straight forward. The main issues I was having was figuring out how to get the data into an object that had the number and the text for each number. I made the object look like this "{2: "Fizz", 5: "Buzz", 10: "Boom"}" for example. Once I had that object it wasn't too hard to loop over it and append the output string to show the correct text.
